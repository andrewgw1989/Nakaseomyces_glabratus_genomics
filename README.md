# Nakaseomyces_glabratus_genomics
Scripts used in genomic analysis of Nakaseomyces glabratus invasive isolates
<br>
<br>
**Source of input files:**
1. FASTQ from Illumina sequencing
2. BAM from BWA within MDU-PHL/Bohra
3. VCF from Snippy with MDU-PHL/Bohra
4. isolates.list file from MDU-PHL/Bohra
